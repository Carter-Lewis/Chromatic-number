# Chromatic number of a graph

## Usage (MacOS):

1. **Build Project:**
    ```bash
    g++ -std=c++11 main.cpp
    ```

2. **Run Project:**
    ```bash
    ./a.out
    ```

## Input Format:
    Enter a number N indicating the number of vertices in the graph
    On the next N lines, enter a pair of characters representing an edge between the two vertices. (ex. a b)
